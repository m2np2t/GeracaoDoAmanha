Título

Pague uma refeição para alguém que precisa.

Descrição

Pague uma refeição para uma pessoa. Pode ser feito até aleatoriamente, mas faça somente se isso não comprometer o seu orçamento.

Justificativa

Esse é um ato de bondade e você estará ajudando uma pessoa.

Processo

Ofereça a gentileza. Se ela aceitar pague a conta ou dê o valor de uma refeição para ela.
